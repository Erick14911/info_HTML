Formularios

  • Etiqueta "form" con atributos 'enctype=text/plain/multipart/form-data/','method=post/get','action=url/correo/'
  • Es importante colocar los atributos identificadores 'name=""' o 'id=""'
  • Etiquetas "input" con atributos
    'type="submit/text/password/radio/checkbox/reset/button/image/hidden/file/email/search/number/range/date/time/datetime-local/color"',
    'value="valor"','size=""','maxlength=""','title="mensaje"','placeholder="texto"','autofocus','disabled','checked',
    'onclick="function"','autocomplete="off/on"','required','multiple'
  • Etiquetas "button" con atributos 'type=reset/'
  • Las etiquetas "button" pueden incluir etiquetas "img" dentro
  • Etiqueta "textarea" con atributos 'rows','cols',wrap="soft/off/hard"La etiqueta list
  • Etiqueta "select" con atributos 'size=""','multiple' con Etiquetas dentro "option" con atributos 'value=""'
  • La etiqueta "datalist" con etiquetas dentro "option" con atributos 'value=""' y 'label="mensaje"' son llamadas por la etiqueta "input" por el atributo 'list='
  • En los "input" tipo=number/range podemos poner los atributos 'min="0" max="10" step="1"'
  • Una etiqueta "input" puede mandarse a un formulario X haciendo referencia con el atributo 'form=""'
  • Uso de la etiqueta "output" para indicar el rango
